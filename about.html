tw-web

A lightweight PWA web interface for Taskwarrior 2.6.x, part of the awesome-taskwarrior fleet.

github.com/linuxcaffe/tw-web

Views

  • Task List — browse, filter, add, edit and action pending tasks
  • Kanban — drag tasks between workflow columns by state UDA
  • Day Planner — drag tasks into hourly time blocks for daily scheduling
  • Calendar — week/month view of scheduled tasks

Firefox Extension

🦊 tw-web Bookmark

Bookmark any page into Taskwarrior with Alt+W. The URL goes into an annotation (clickable in tw-web). Also opens the Add task dialog directly.
